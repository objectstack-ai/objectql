## What is ObjectQL?

**ObjectStack ObjectQL** is a production-ready protocol and engine for building metadata-driven applications. It decouples **Intent** (Schema & Logic) from **Implementation** (Database & Code).

By defining your data models, validation rules, and business logic in simple standard formats (YAML/JSON), ObjectQL compiles them into a type-safe backend with automatic validation and formula evaluation.

## Application-Layer Features

These features require implementation in your application code:

- **Permissions/RBAC** - Implement using hooks or middleware
- **Workflows** - Build using state machine validation + hooks  
- **Audit Trails** - Track changes using hooks
- **Reports** - Generate using query API + external libraries
